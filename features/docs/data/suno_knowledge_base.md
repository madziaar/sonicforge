## Prompting Best Practices

### The Golden Rules
1. **Be Specific**: "Alternative indie rock" > "rock song"
2. **Use Custom Mode**: Better control and consistency
3. **Meta Tags**: `[Intro]`, `[Verse]`, `[Chorus]` guide structure
4. **Order Matters**: Genre -> Mood -> Tempo -> Vocals -> Instruments

### Effective Prompt Structure
```
Primary Genre, Secondary Genre, Mood, Tempo, Key,
Vocal Style, Primary Instruments, Atmospheric Elements,
Production Quality
```

## Meta Tags Reference

### Structural
`[Intro]`, `[Verse]`, `[Chorus]`, `[Bridge]`, `[Outro]`, `[Instrumental]`

### Instrumental
`[Guitar Solo]`, `[Piano Solo]`, `[Sax Solo]`, `[Drum Break]`, `[Bass Drop]`
*Pro Tip: Use repeated, lowercase tags: `[sax][saxophone][solo]`*

### Vocal
`[Female Vocal]`, `[Male Vocal]`, `[Whispered]`, `[Screaming]`, `[Auto-tune]`

### Ending (Critical)
`[Outro]`, `[Instrumental Fade out]`, `[End]`

## Troubleshooting Guide

**Abrupt Song Endings?**
Add `[Instrumental Fade out][End]` to the lyrics.

**Wrong Instrument?**
Use repeated tags like `[sax][saxophone][solo]` and keep descriptions concise.

**Robotic Vocals?**
Upgrade to v5 or add detailed vocal descriptors like "Emotional female vocals, natural delivery".

**British Accent in Rap?**
Specify regional style: "Memphis trap", "Atlanta trap", "West Coast". Use "Phonk Drum" tag.
