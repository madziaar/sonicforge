## Style Prompt Mastery

### The Perfect Formula

```
[GENRE] + [MOOD] + [TEMPO] + [VOCALS] + [INSTRUMENTS] + [PRODUCTION]
```

### Element Priority Order

**1. Genre (REQUIRED)** - Most important
- Be specific: "Synthwave" not "electronic"
- Use sub-genres: "Memphis trap" not "trap"
- Examples: Heavy Metal, Deep House, Indie Rock, Boom Bap

**2. Mood (HIGHLY RECOMMENDED)** - 1-3 descriptors
- Dark, Upbeat, Melancholic, Energetic, Chill, Aggressive
- Don't mix opposites: not "energetic, calm"
- Layer compatible moods: "dark, atmospheric, brooding"

**3. Tempo (OPTIONAL BUT HELPFUL)**
- Specific BPM: "140 BPM", "122 BPM"
- OR descriptors: "driving tempo", "laid-back", "frenetic"
- Match genre conventions:
  - Trap: 130-150 BPM
  - House: 120-130 BPM  
  - Metal: 120-180 BPM
  - Lo-fi: 80-95 BPM

**4. Vocals (HIGHLY RECOMMENDED)**
- Gender: Male, Female, Androgynous
- Quality: Clean, Harsh, Ethereal, Gritty, Smooth
- Delivery: Powerful, Gentle, Aggressive, Whispered
- Regional: UK accent, Southern flow, West Coast
- Example: "Ethereal female vocals, breathy delivery"

**5. Instruments (RECOMMENDED)** - 3-5 key ones
- Be specific: "Jangly guitars" not "guitars"
- Include playing style: "Fingerpicked acoustic guitar"
- For electronic: "808 bass", "vintage synthesizers"
- For rock: "distorted guitars", "driving drums"

**6. Atmosphere/Texture (OPTIONAL)** - 1-3 descriptors
- Warm, Cold, Spacious, Intimate, Raw, Polished
- Lush, Sparse, Dense, Airy, Gritty
- Example: "Warm analog atmosphere, spacious mix"

**7. Production (OPTIONAL)** - 1-2 keywords
- Studio-quality, Mastered, Lo-fi, Raw
- Clean mix, Analog warmth, Modern production
- Example: "Studio-quality production, clear mix"

**Mood Families:**
- Dark family: mysterious, ominous, brooding, noir
- Happy family: joyful, euphoric, uplifting, cheerful
- Sad family: melancholic, somber, emotional, wistful
- Energetic family: driving, powerful, explosive, frenetic

- Metal: Guttural, harsh, screaming, growling

- Modern: Polished, clean mix, mastered, studio-quality

## Lyrics Formatting

### Meta Tag Essentials

**What are Meta Tags?**
- Instructions in square brackets: `[Verse]`, `[Chorus]`
- Guide Suno's AI on song structure
- Control vocals, instruments, transitions
- MUST be in square brackets `[]`

**Where to Place:**
- On their own line
- Before the section starts
- Can stack: `[Verse 1][Male Vocal]`

### Basic Song Structure

**Pop/Rock Standard:**
```
[Intro]
[Instrumental]

[Verse 1]
8 lines of lyrics

[Pre-Chorus]
4 lines building tension

[Chorus]
8 lines, main hook

[Verse 2]
8 lines of lyrics

[Pre-Chorus]
4 lines building tension

[Chorus]
8 lines, main hook

[Bridge]
6-8 lines, emotional shift

[Chorus]
8 lines, final hook

[Outro]
[Instrumental Fade out][End]
```

**Hip Hop Structure:**
```
[Intro]
[Ad-libs or instrumental]

[Verse 1]
16 bars

[Hook]
8 bars

[Verse 2]
16 bars

[Hook]
8 bars

[Bridge]
8 bars (optional)

[Hook]
8 bars

[Outro]
[Ad-libs][Fade out]
```

**Electronic Structure:**
```
[Intro]
[Atmospheric build]

[Build-up]
[Increasing tension]

[Drop]
[Main energy section]

[Break]
[Calm section]

[Build-up]
[Increasing tension]

[Drop]
[Final climax]

[Outro]
[Fade out]
```

### Ending Your Song Properly

**Problem:** Songs often cut off abruptly without proper endings.

**Solutions:**

**Option 1: Clean Ending**
```
[Outro]
[End]
```

**Option 2: Fade Out**
```
[Outro]
[Fade out]
```

**Option 3: Best (Most Natural)**
```
[Outro]
[Instrumental Fade out][End]
```

**Option 4: Extended Outro**
```
[Outro]
Final lyrics here
[Instrumental]
[Fade out]
```

## Meta Tags Complete Reference

### Structural Tags (Essential)

| Tag | Purpose | Usage |
|-----|---------|-------|
| `[Intro]` | Song introduction | Start of song |
| `[Verse]` | Main story sections | `[Verse 1]`, `[Verse 2]` |
| `[Pre-Chorus]` | Build before chorus | Before chorus |
| `[Chorus]` | Main hook/refrain | Repeated section |
| `[Bridge]` | Contrasting section | After 2nd chorus |
| `[Outro]` | Song ending | End of song |

### Vocal Tags

| Tag | Effect |
|-----|--------|
| `[Female Vocal]` | Female voice |
| `[Male Vocal]` | Male voice |
| `[Ethereal Voice]` | Dreamy, heavenly |
| `[Powerful Vocals]` | Strong, loud |
| `[Whispered]` | Soft, intimate |
| `[Screaming]` | Intense, aggressive |
| `[Clean vocals]` | Clear singing |
| `[Harsh vocals]` | Metal/aggressive |
| `[Auto-tune]` | Auto-tuned effect |

### Instrumental Tags

| Tag | Effect |
|-----|--------|
| `[Instrumental]` | No vocals section |
| `[Guitar Solo]` | Guitar solo |
| `[Piano Solo]` | Piano solo |
| `[Sax Solo]` | Saxophone solo |


## Advanced Techniques

### 1. Vowel Extension (Melodic Vocals)

**Creates longer, more melodious vocal lines**

**Syntax:** Extend vowel with hyphens
```
goodbye ‚Üí goo-o-o-odbye
love ‚Üí lo-o-o-ove
stay ‚Üí sta-a-a-ay
free ‚Üí fre-e-e-ee
```

**Example:**
```
[Chorus]
I'll ne-e-e-ever let you go-o-o-o
This lo-o-o-ove will always gro-o-o-ow
```

**Best for:**
- Choruses
- Emotional moments
- Sustained notes
- Pop/R&B vocals

### 2. Background Vocals (Parentheses)

**Adds backing vocals automatically**

**Syntax:** Put background text in parentheses
```
Main lyric (background response)
```

**Examples:**
```
E la cha-cha-cha (cha)
Feel the rhythm (feel it)
Can't stop dancing (dancing tonight)
Yeah yeah (oh yeah)
```

**Creates:**
- Call and response
- Vocal layers
- Depth and richness
- Choir effects

### 3. Chord Annotations

**Guides harmonic structure**

**Syntax:** Chord names in parentheses before words
```
(Em) Walking down the street
(G) Under city lights
(D) Searching for a sign
(C) Everything's alright
```

**Example Progression:**
```
[Verse 1]
(Am) The night is dark and cold
(F) Whispers in the air
(C) Shadows dance around
(G) In this place of despair
```

**Benefits:**
- Controls melody direction
- Guides harmonic choices
- More musical control
- Better for specific vibes

### 4. Musical Notation Per Syllable

**Extreme melodic control**

**Syntax:** Note name before each syllable
```
(G)Beat (G)of (A)the (B)heart (C)goes (D)on
```

**Example:**
```
[Chorus]
(C)Walk (D)ing (E)down (F)the (G)street (A)to (B)night (C)
```

**Use when:**
- Need exact melody
- Specific note progression
- Complex musical ideas
- Advanced productions

### 5. Inline Style Descriptions

**Section-specific styling**

**Syntax:**
```
(Section Name) [style description]:
Lyrics here
```

**Examples:**
```
(Chorus) [heavy riffs, driving beat, powerful vocals]:
We're unstoppable tonight
Fighting through the dark

(Verse 1) [soft synths, gentle vocals, minimal drums]:
Whispers in the morning light
Shadows slowly fade away

(Bridge) [stripped back, piano only, emotional delivery]:
And all we have is now
This moment here with you
```

**Benefits:**
- Fine-tune each section
- Dynamic arrangements
- Professional control
- Varied soundscapes

## Common Mistakes to Avoid

### Style Prompt Mistakes

‚ùå **Too Generic**
"rock music" ‚Üí ‚úÖ "Alternative Indie Rock"

‚ùå **Too Long/Cluttered**
"Rock music with guitars and drums and bass and vocals and nice melodies..." 
‚Üí ‚úÖ "Rock, Energetic, Guitar-driven, Melodic"

‚ùå **Conflicting Descriptors**
"Energetic, calm, dark, bright" 
‚Üí ‚úÖ Pick compatible moods: "Dark, atmospheric, brooding"

‚ùå **Missing Essentials**
"128 BPM, male vocals" 
‚Üí ‚úÖ Must include genre: "Indie Rock, 128 BPM, Male vocals"

‚ùå **No Specificity**
"Electronic music" 
‚Üí ‚úÖ "Deep House, 122 BPM, Groovy, Sub bass"

‚ùå **Missing Section Tags**
```

```
‚Üí ‚úÖ Use tags:
```

[Chorus]
Chorus lyrics
```

‚ùå **No Ending Tag**
```
[Chorus]

```
‚Üí ‚úÖ Add ending:

‚ùå **Unclosed Brackets**
```

```


‚ùå **Wrong Bracket Type**

## Pro Tips

### Style Prompts

**1. Start with Genre-Specific Templates**
- Don't start from scratch
- Use proven templates
- Customize to your needs

**2. Layer Moods Carefully**
- 2-3 compatible moods max
- "Dark, atmospheric, brooding" ‚úÖ
- "Happy, sad, energetic" ‚ùå

**3. Match Genre Conventions**
- Trap: 130-150 BPM, 808s, hi-hats
- House: 120-130 BPM, 4/4 beat
- Metal: Minor keys, distorted guitars

**4. Specify Vocal Details**
- Not just "female vocals"
- "Ethereal female vocals, breathy delivery" ‚úÖ

**5. Production Quality Matters**
- Add: "Studio-quality", "Mastered", "Clear mix"
- Or: "Lo-fi", "Raw", "Analog warmth"

### Lyrics Formatting

**1. Always Use Structure Tags**
- Never leave lyrics untagged
- Clear `[Verse]`, `[Chorus]` labels
- Helps Suno understand flow

**2. Include an Ending**
- `[Outro][Fade out]` is reliable
- Songs without endings cut abruptly

**3. Keep Sections Balanced**
- Verses: 8 lines
- Choruses: 6-8 lines
- Bridges: 6-8 lines

**4. Repeat Chorus**
- Chorus should appear 3+ times
- Helps create memorable songs

**5. Use Advanced Techniques Sparingly**
- Vowel extension: 2-3 spots max
- Background vocals: Key moments
- Don't overuse in every line

### General Workflow

**1. Choose Genre First**
- Everything else follows from genre
- Genre determines BPM, instruments, style

**2. Use Enhancement**
- Start simple, enhance to detailed
- "trap" ‚Üí "Memphis trap, Dark, 140 BPM..."

**3. Generate Variations**
- Create 3-5 different approaches
- Test what works best in Suno

**4. Save Successful Prompts**
- Keep library of what works
- Reuse and modify proven templates

**5. Iterate Based on Results**
- First generation rarely perfect
- Adjust prompt based on output
- Fine-tune until satisfied
