## Quick Start Guide (v5 + Studio)

### For Beginners (Free/Pro)

1. Sign up at suno.com
2. Use Simple Mode: "Create a [genre] song about [topic]"
3. Let AI handle everything
4. Download (while still available)

### For Creators (Pro)

1. Use Custom Mode
2. Write detailed style prompt:
   ```
   Genre, Sub-genre, Mood, BPM, Key,
   Vocal style, Instruments, Production
   ```
3. Add your lyrics with meta tags
4. Use v5 for best quality
5. Extend or remix as needed

### For Professionals (Premier)

1. Open Suno Studio
2. **Start with:**
   - Upload existing audio, or
   - Generate from prompt, or
   - Import from Suno library
3. **Edit in timeline:**
   - Generate stem variations
   - Reorder sections
   - Adjust creative sliders
4. **Extract stems** (12 channels)
5. **Export:**
   - Audio files for distribution
   - MIDI for further production
   - Stems for mixing in your DAW
